# Cain and Abel - A Journey of Unity (ProWeaver Hackathon PromptQuest 2025 Entry)

## Project Title

**Cain and Abel - A Journey of Unity**

## Description

"Cain and Abel - A Journey of Unity" is a 2D cooperative puzzle platformer game developed as an entry for the ProWeaver Hackathon PromptQuest 2025. This remastered version reimagines the classic biblical narrative, transforming it into an engaging challenge where two brothers, each with unique abilities, must work together to overcome obstacles and solve environmental puzzles. The game explores themes of cooperation, sacrifice, and ultimately, confrontation.

## Features

*   **Dual Character Gameplay:** Play as two distinct characters, Cain (Earth-Bound) and Abel (Sky-Tender), each possessing specialized skills.
*   **Unique Character Abilities:**
    *   **Cain (Earth-Bound):** Can push heavy objects, create temporary earth platforms, and break through physical barriers.
    *   **Abel (Sky-Tender):** Possesses enhanced agility with double-jump and glide abilities, and can illuminate dark areas to reveal hidden paths and switches.
*   **Dynamic Puzzles:** Navigate through levels requiring strategic use of both characters' abilities, from bridge-building and barrier-breaking to illuminating hidden elements.
*   **Immersive Narrative:** Experience a reimagined take on the Cain and Abel story, culminating in a dramatic final confrontation.
*   **Remastered Visuals & UI:**
    *   Modernized dark-themed UI with a glass-morphism aesthetic.
    *   Thematic typography inspired by historical and biblical texts.
    *   Enhanced in-game graphics with particle effects for movements and abilities (dust, sparkles, debris).
    *   Dynamic screen shake effects for impactful actions.
    *   Atmospheric lighting system with illuminated areas and overall darkness.
*   **Responsive Design:** Adapts to various screen sizes, ensuring playability on different devices.
*   **Atmospheric Audio:** Incorporates background music that shifts with the game's progression and narrative tone.

## How to Play

### Controls

**Cain (Earth-Bound):**
*   **WASD:** Move
*   **SPACE:** Jump
*   **E:** Push/Pull objects
*   **R:** Create Earth Platform (temporary)
*   **T:** Break Barrier

**Abel (Sky-Tender):**
*   **J/L:** Move (Left/Right)
*   **SHIFT:** Jump / Double Jump
*   **F:** Illuminate Area (reveals hidden elements)

**Game System Controls:**
*   **1 / 2:** Switch Character (1 for Cain, 2 for Abel)
*   **P:** Pause Game
*   **Ctrl + R:** Restart Current Level

### Objective

Guide both Cain and Abel through each level, utilizing their combined strengths to overcome obstacles. Collect all offerings and reach the exit gate to proceed. The final level presents a pivotal narrative choice.

## Installation / Setup

This game is a client-side web application and requires no special installation.

1.  **Download:** Clone or download the project files to your local machine.
2.  **Open:** Navigate to the project directory and open `index.html` in your preferred web browser (e.g., Chrome, Firefox, Edge).
3.  **Play:** Click the "Begin Journey" button to start the game.

## Technologies Used

*   **HTML5:** Structure and content.
*   **CSS3:** Styling, animations, and responsive layout.
*   **JavaScript (ES6+):** Core game logic, physics engine, rendering, and interactive elements (vanilla JavaScript, no external frameworks/libraries).
*   **Canvas API:** For all in-game rendering.
